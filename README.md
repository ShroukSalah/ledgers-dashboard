# Ledgers Analytics Dashboard

A responsive analytics dashboard built as part of the **Ledgers Frontend Developer Technical Assignment**.

The project focuses on **pixel-accurate UI implementation**, clean **Vue 3 component architecture**, and efficient **Tailwind CSS** usage while maintaining a fully responsive layout across **desktop, tablet, and mobile devices**.

---

## Tech Stack

* Vue 3
* Vite
* Tailwind CSS
* PrimeVue
* Chart.js

---

## Features

* Responsive analytics dashboard layout
* Reusable Vue components
* Clean component architecture
* Tailwind utility-first styling
* Interactive charts using PrimeVue / Chart.js
* Modular structure using Vue composables

---

```
src
```

## Installation

Install dependencies:

```
npm install
```

---

## Run Development Server

```
npm run dev
```

Then open:

```
http://localhost:5173
```
