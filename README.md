# ledgers-dashboard
Ledgers Analytics Dashboard  A responsive analytics dashboard built as part of the Ledgers Frontend Developer Technical Assignment.  The project focuses on pixel-accurate UI implementation, clean Vue 3 component architecture, and efficient Tailwind CSS usage, while maintaining a fully responsive layout across desktop, tablet, and mobile devices.
## Stack
Vue 3
Vite
Tailwind CSS
PrimeVue

## Install

npm install

## Run

npm run dev

## Features

- Responsive analytics dashboard
- Reusable Vue components
- Tailwind utility-first styling
- PrimeVue charts
